    <h1>Movie Recommendation System</h1>
    <h2>Overview</h2>
    <p>
        This project recommends movies to users based on their viewing history and preferences using collaborative filtering.
        The dataset includes user ratings for movies, which are used to find similar users and recommend top-rated unseen movies.
    </p>
    <h2>Features</h2>
    <ul>
        <li><strong>User-Based Collaborative Filtering:</strong> Finds similar users and recommends movies they liked.</li>
        <li><strong>Item-Based Filtering:</strong> (Bonus) Recommends movies similar to those the user has already enjoyed.</li>
        <li><strong>Matrix Factorization:</strong> Implements SVD to improve recommendation quality.</li>
        <li><strong>Evaluation:</strong> Uses precision@K for performance measurement.</li>
        <li><strong>Data Handling:</strong> Builds and manages user-item rating matrices efficiently.</li>
    </ul>
